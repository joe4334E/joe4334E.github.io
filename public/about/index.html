About FeelIt

Features

Performance and SEO

  • :(fa-solid fa-rocket fa-fw): Optimized for performance: [99]/[100] on mobile and [100]/[100] on desktop in Google PageSpeed Insights
  • :(fa-brands fa-searchengin fa-fw): Optimized SEO performance with a correct SEO SCHEMA based on JSON-LD
  • :(fa-brands fa-google fa-fw): Google Analytics supported
  • :(fa-regular fa-chart-bar fa-fw): Fathom Analytics supported
  • :(fa-solid fa-chart-line fa-fw): GoatCounter Analytics supported
  • :(fa-solid fa-search fa-fw): Search engine verification supported (Google, Bind, Yandex and Baidu)
  • :(fa-solid fa-tachometer-alt fa-fw): CDN for third-party libraries supported
  • :(fa-solid fa-cloud-download-alt fa-fw): Automatically converted images with Lazy Load by lazysizes

Appearance and Layout

  • [:(fa-solid fa-desktop):]/[:(fa-solid fa-mobile):] Responsive layout
  • [:(fa-solid fa-sun):]/[:(fa-solid fa-moon):] [Light]/[Dark] mode
  • :(fa-solid fa-layer-group fa-fw): Globally consistent design language
  • :(fa-solid fa-ellipsis-h fa-fw): Pagination supported
  • :(fa-regular fa-list-alt fa-fw): Easy-to-use and self-expanding table of contents
  • :(fa-solid fa-language fa-fw): Multilanguage supported and i18n ready
  • :(fa-brands fa-css3-alt fa-fw): Beautiful CSS animation

Social and Comment Systems

  • :(fa-regular fa-user fa-fw): Gravatar supported by Gravatar
  • :(fa-solid fa-user-circle fa-fw): Local Avatar supported
  • :(fa-regular fa-id-card fa-fw): Up to 64 social links supported
  • :(fa-solid fa-share-square fa-fw): Up to 28 share sites supported
  • :(fa-regular fa-comment fa-fw): Disqus comment system supported by Disqus
  • :(fa-regular fa-comment-dots fa-fw): Gitalk comment system supported by Gitalk
  • :(fa-regular fa-comment-alt fa-fw): Valine comment system supported by Valine
  • :(fa-solid fa-comments fa-fw): Vssue comment system supported by Vssue
  • :(fa-regular fa-comments fa-fw): Facebook comments system supported by Facebook
  • :(fa-solid fa-comment fa-fw): Telegram comments system supported by Comments
  • :(fa-solid fa-comment-dots fa-fw): Commento comment system supported by Commento
  • :(fa-solid fa-comment-alt fa-fw): Utterances comment system supported by Utterances
